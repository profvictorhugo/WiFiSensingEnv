Este projeto foi criado utilizando o **Vite** para construção de uma aplicação React moderna. O objetivo é fornecer uma base inicial bem organizada para projetos com múltiplas páginas e CRUDs.

## **Instalação e Configuração**

Siga os passos abaixo para clonar, instalar as dependências e rodar o projeto localmente.

### **Requisitos Pré-requisitos**

- **Node.js** (>= 16.0.0): [Baixar aqui](https://nodejs.org/).
- Gerenciador de pacotes **npm** (já incluído com o Node.js).

### **Passos para Instalar**

1. **Clone o repositório:**

   ```bash
   git clone <URL_DO_REPOSITORIO>
   cd frontend
   ```

2. **Instale as dependências do projeto:**

   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**

   ```bash
   npm run dev
   ```

4. **Abra o navegador:**
   O projeto será iniciado em `http://localhost:5173`.

## **Scripts Disponíveis**

- **`npm run dev`**: Inicia o servidor de desenvolvimento.
- **`npm run build`**: Cria a versão de produção.
- **`npm run preview`**: Visualiza o build de produção localmente.
